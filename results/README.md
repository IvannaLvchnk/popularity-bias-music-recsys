# Results

This folder contains summary tables generated by the notebook.

## Files

| File | Description |
|---|---|
| `bias_on_recommendations.csv` | ARP, ACLT, Exposure Gini, Catalog Coverage for all three baseline models |
| `mitigation_results.csv` | Full λ-sweep results for xQuAD and FairRerank (NDCG, ACLT, Coverage) |
| `raw_bias_summary.csv` | Artist-side and genre-side Gini + top-20% statistics for LFM-1b and XITE |

These files are generated by running `notebooks/popularity_bias_FULL_v5.ipynb`
and are committed here for convenience so the presentation and README numbers
can be verified without re-running the full pipeline.
